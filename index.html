About Section Slot

關於我（之後會用 Section SDD 替換）

這裡目前是 About 區的占位內容，用來說明個人背景、職涯定位與前端開發相關經歷。實際上線前，會用另一個專門針對 About 區的 Section SDD Prompt 產生更完整的敘事與版面配置。

未來可以在這一區放入「自我介紹、職涯轉職故事、專注領域」等內容，例如從非本科轉職成前端工程師的歷程、對使用者體驗與程式品質的堅持、以及未來希望合作的專案類型。這些文字目前都只是說明用的占位，主要目的是讓版型與結構先建立起來。

開發者在之後使用 Section SDD Prompt 時，請直接產生一整個 <section id="about" ...>...</section> 的完整程式碼，然後將產出的整個 section 貼回到 SECTION:ABOUT-START / SECTION:ABOUT-END 註解之間，完全覆蓋現在這一區的內容，無需保留現有段落。

提醒：之後替換時，只需要關心這一個 section 區塊即可，無須修改其他區塊與 CSS Slot，能降低出錯機率，維持整體版型穩定。

給開發者的占位說明

· 這裡是 About 區的 HTML 骨架插槽。
· 之後會用 Section SDD 的 Prompt 產生新的 About 區。
· 請把新的 section 程式碼整段貼在 SECTION:ABOUT-START / SECTION:ABOUT-END 兩個註解中間，完全取代現在的區塊。
· 若 Section SDD 有額外的 class 命名，可在 CSS:ABOUT Slot 中追加或覆寫對應樣式。
